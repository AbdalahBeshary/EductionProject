# Restaurant Ordering Web Application

A real-world web application created using Lovable AI that allows customers to browse a restaurant menu and place orders online.

## Description
This project demonstrates the ability to use AI-assisted web tools to build practical, customer-facing web applications.

## Key Features
- Online menu display
- Order placement functionality
- Simple and clear user interface

## Note
This project is included as a non-educational example to demonstrate real-world web application development skills.
